Timestamp API

Gets the UNIX timestamp and natural language date for a specified date.

The date can be specified as a UNIX timestamp or a natural language date string.

Response properties will be null if the specified date is invalid.

URL

GET https://twisty-hour.glitch.me/<date>

Example usage:

https://twisty-hour.glitch.me/December%2015,%202015

https://twisty-hour.glitch.me/1450137600

Example response:

{

"natural": "December 15, 2015"

"unix": 1450137600,

}
